# Desmos

Bond every link.

Desmos is a cross-platform connection bonding VPN written in Rust. It combines
multiple network interfaces (Wi-Fi, Ethernet, LTE, and more) into a single
encrypted tunnel that is simultaneously faster and more reliable than any
individual link.

Status: pre-alpha. Not yet usable. See `TASKS.md` for the implementation plan.
